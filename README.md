# Project-Excel-Interactive-Dashboard-For-Excel
**Interactive Excel Dashboard** is a data visualization project built in Microsoft Excel to analyze sales and performance data. It uses Pivot Tables, Charts, and Slicers for dynamic filtering and insights. This dashboard helps track sales, profit, and product performance in an easy and interactive way, improving data analysis and reporting skills.
Interactive Excel Dashboard Project Description

Project Title: Sales & Performance Interactive Dashboard in Excel

This project is an interactive Excel dashboard designed to analyze and visualize business data efficiently. It helps in tracking key performance indicators (KPIs) such as total sales, profit, quantity sold, and product performance.

The dashboard uses advanced Excel features like:

Pivot Tables for data summarization
Pivot Charts for visual analysis
Slicers for interactive filtering
Conditional Formatting for highlighting important insights
Formulas & Functions for automated calculations
Key Features:

✔ Dynamic filtering by month, product, and category
✔ Sales trend analysis
✔ Top-performing products tracking
✔ Profit and revenue visualization
✔ User-friendly and interactive interface

Tools Used:
Microsoft Excel
Pivot Tables
Charts
Slicers
Data Cleaning Techniques
Purpose of Project:

The main goal of this dashboard is to improve data analysis skills and provide a simple business intelligence solution for decision-making.
